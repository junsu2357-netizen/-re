CHAPTER A클라우드 기초 개념
🏢클라우드 (Cloud)CLOUD▼
인터넷 너머 어딘가에 있는 거대한 컴퓨터 창고
예전에는 회사가 직접 서버(컴퓨터)를 사서 방 한 칸을 채워야 했습니다. 클라우드는 이걸 매달 요금을 내고 빌려 쓰는 것입니다.

📦 비유: 창고를 직접 짓는 대신 → 창고 임대 서비스를 이용하는 것필요할 때 크게, 한산할 때 작게 조절할 수 있어 비용과 유연성 모두 이득입니다. AWS(아마존), Azure(마이크로소프트), GCP(구글)이 세계 3대 클라우드입니다.
AWSAzureGCP
🖥️온프레미스 (On-Premise)ON-PREM▼
회사 건물 안에 직접 서버를 두는 방식
클라우드의 반대 개념으로, 회사가 직접 서버 장비를 구매하고 유지·관리합니다.

🏠 비유: 집을 직접 사서 관리하는 것 ↔ 월세(클라우드)초기 비용이 크지만 데이터가 회사 밖으로 나가지 않아 보안에 민감한 금융·공공기관이 선호합니다.
🔀하이브리드 클라우드HYBRID CLOUD▼
회사 내부 서버 + 클라우드를 함께 사용하는 방식
중요한 데이터는 회사 내부에, 유연성이 필요한 부분은 클라우드에 올리는 혼합 방식입니다.

🍱 비유: 집밥(온프레미스) + 배달음식(클라우드)을 상황에 따라 선택보안과 유연성을 동시에 챙길 수 있어 대기업·금융기관에서 많이 사용합니다. 메타넷의 핵심 사업 영역입니다.
📋IaaS / PaaS / SaaSSERVICE MODELS▼
클라우드 서비스의 3가지 제공 방식
IaaS (Infrastructure as a Service): 서버·네트워크만 빌려줌 → 건물 뼈대만 제공
PaaS (Platform as a Service): 개발 환경까지 빌려줌 → 인테리어까지 된 공간
SaaS (Software as a Service): 완성된 소프트웨어 자체를 빌려줌 → 꾸며진 집에 바로 입주

📌 예시: IaaS=AWS EC2, PaaS=Google App Engine, SaaS=Gmail·Netflix
🗂️멀티 클라우드MULTI CLOUD▼
AWS, Azure, GCP 등 여러 클라우드를 동시에 쓰는 전략
하나의 회사가 AWS, Azure, 네이버 클라우드를 동시에 사용하는 전략입니다.

🛒 비유: 쿠팡·11번가·네이버쇼핑을 비교하며 최저가 주문하는 것특정 클라우드에 의존하지 않아 장애 위험이 줄고, 각 클라우드의 장점을 골라 쓸 수 있습니다.
🚚클라우드 마이그레이션MIGRATION▼
기존 IT 시스템을 클라우드로 이사시키는 작업
회사가 오래 쓰던 서버와 시스템을 클라우드로 옮기는 것입니다. 통째로 옮기거나(Lift & Shift), 이사하는 김에 재설계하는 방법 등 다양한 전략이 있습니다.

🏚️→🏠 비유: 낡은 집에서 새 아파트로 이사하는 것메타넷은 NCP/VMware → AWS 같은 복잡한 이사 작업을 전문으로 수행합니다.
7R 전략Lift & ShiftRe-architecture

CHAPTER BMSP & 클라우드 운영
🏭MSP (Managed Service Provider)MSP▼
클라우드를 대신 관리해주는 전문 회사
기업이 클라우드를 구축하고 운영하는 전 과정을 대행해주는 IT 서비스 회사입니다. 24시간 365일 모니터링, 비용 최적화, 장애 대응까지 맡아줍니다.

🧑‍🔧 비유: 건물 관리인 — 집주인(기업) 대신 건물(클라우드)의 모든 것을 관리메타넷은 AWS Advanced Tier MSP 파트너로, 삼성전자·KB국민은행 등의 클라우드를 관리합니다.
🔃DevOpsDEVOPS▼
개발팀과 운영팀이 한 팀처럼 움직이는 방식
Development(개발) + Operations(운영)의 합성어입니다. 두 팀이 긴밀하게 협력해 빠르고 안정적으로 소프트웨어를 배포하는 문화·방식입니다.

🏭 비유: 요리사(개발)와 홀 직원(운영)이 함께 메뉴 개선을 논의하는 레스토랑
CI/CDGitLabJenkins
🔄CI/CD (지속적 통합/배포)CI/CD▼
코드 변경사항을 자동으로 테스트하고 배포하는 파이프라인
CI (Continuous Integration): 코드를 합칠 때마다 자동으로 테스트 → 오류 즉시 발견
CD (Continuous Deployment): 테스트 통과한 코드를 자동으로 서비스에 배포

🏃 비유: 음식 주문 → 자동 조리 → 자동 배달까지 전 과정 자동화메타넷은 GitLab, Jenkins, AWS CodeDeploy를 활용합니다.
📜IaC (Infrastructure as Code)IaC▼
서버 인프라 설정을 코드로 자동화하는 방법
서버 100대를 손으로 세팅하는 대신, 코드 한 번 실행으로 자동 처리합니다.

🖨️ 비유: 집 설계도(코드)만 있으면 버튼 하나로 집 1000채를 자동 건설Terraform, Ansible이 대표 도구입니다.
TerraformAnsible
📦컨테이너 (Container)CONTAINER▼
앱을 독립된 박스에 넣어 어디서나 똑같이 실행시키는 기술
"내 컴퓨터에선 되는데 서버에선 안 돼요" 문제를 해결합니다. 앱 실행에 필요한 모든 것을 하나의 박스(컨테이너)에 담아 어떤 환경에서도 동일하게 실행됩니다.

🚢 비유: 국제 표준 컨테이너 — 배에도, 트럭에도, 기차에도 올릴 수 있는 표준 박스
DockerKubernetesOpenShift
🎛️Kubernetes (쿠버네티스)K8S▼
수천 개의 컨테이너를 자동으로 관리·배분하는 시스템
컨테이너가 수백~수천 개가 되면 혼자 관리 불가능합니다. Kubernetes는 컨테이너들을 자동 배분하고, 하나가 죽으면 새걸 띄워주고, 트래픽이 몰리면 수를 늘리는 자동 관리 시스템입니다.

🕹️ 비유: 수천 명의 직원(컨테이너)에게 자동으로 업무를 배분하는 AI 인사팀장구글이 만들었으며, "K8s"라고도 부릅니다.

CrowdStrikeCybereasonMDE

🔌 비유: 수천 개의 수도꼭지(로그)를 하나의 대형 파이프로 연결해 저수지(저장소)로 보내는 것메타넷의 AWS 보안관제 아키텍처에서 핵심 로그 수집 역할을 담당합니다.

🗄️서버 / 스토리지SERVER / STORAGE▼

스토리지: 페타바이트(1,000TB) 단위 기업 데이터를 저장하는 대용량 창고.

🍽️ 서버 비유: 손님(사용자)의 주문(요청)을 처리해 서빙하는 레스토랑 주방

🏗️HPC (고성능 컴퓨팅)HPC▼
AI 학습·시뮬레이션·기상예측 등 엄청난 계산을 처리하는 컴퓨팅
High Performance Computing의 약자입니다. 수천 개의 CPU와 GPU를 연결해 일반 컴퓨터로는 수백 년 걸릴 계산을 몇 시간 만에 처리합니다.

🏛️데이터센터 (IDC)IDC / DC▼
수천~수만 대의 서버가 모여 있는 전용 건물
냉방, 전력 이중화, 소화 설비, 물리 보안 등을 갖춘 서버 전용 건물입니다.

🏭 비유: 서버들의 아파트 단지 — 24시간 전기, 냉방, 보안이 완비된 건물클라우드의 등장으로 기업들이 IDC를 직접 운영하는 대신 클라우드를 빌려 쓰는 추세입니다.

서버 가상화의 업계 표준 소프트웨어
vSphere: 물리 서버 위에 VM을 만드는 핵심 가상화 엔진
vCenter: 여러 VMware 서버를 한 화면에서 통합 관리하는 도구
vSAN: 여러 서버의 하드디스크를 하나로 묶어 가상 스토리지를 만드는 기능

🚗 비유: Kubernetes = 자동차 엔진 / OpenShift = 에어백·네비·편의기능을 더한 완성차
🔧Ansible (앤서블)ANSIBLE▼
수천 대 서버 설정을 코드 한 줄로 자동화하는 도구
Red Hat이 만든 IT 자동화 도구입니다. 서버 1,000대에 새 소프트웨어 설치를 코드 한 번 실행으로 동시 처리합니다.

HashiCorp가 만든 IaC 도구입니다. AWS, Azure, GCP 등 여러 클라우드를 하나의 코드로 통합 관리할 수 있습니다.

📐 비유: 건축 설계도(Terraform 코드) → 시공사(클라우드)에 던지면 자동 건설"서버 10대 필요해" 라고 쓰면 Terraform이 알아서 만들어줍니다.

Hyper-Converged Infrastructure의 대표 브랜드입니다. 예전에는 서버, 스토리지, 네트워크 장비를 따로 사야 했지만, Nutanix는 이를 하나의 박스로 통합합니다.

📱 비유: 카메라+전화+음악+내비가 하나로 합쳐진 스마트폰설치·관리가 간단하고 노드(박스)를 추가해 쉽게 확장할 수 있습니다.

🛂 비유: 공항 탑승구 — 공항 안에 있어도 탑승권 없으면 비행기 못 탐재택근무가 많아진 요즘 필수 보안 솔루션입니다.

🏭 비유: 회사 전체 부서의 업무를 하나의 앱으로 연결하는 것메타넷은 SAP를 클라우드(Azure, AWS)로 이전하는 프로젝트를 다수 수행합니다.

CHAPTER G프로젝트 관리 & 비즈니스 용어
📐아키텍처 (Architecture)ARCHITECTURE▼
IT 시스템의 전체 설계도
서버, 네트워크, 데이터베이스, 보안 등 IT 시스템의 각 요소가 어떻게 연결되고 동작하는지를 나타낸 전체 그림입니다.

🏗️ 비유: 집 짓기 전의 건축 설계도 — 어디에 방이 오고, 어떻게 연결되는지 미리 계획메타넷의 SA(Solution Architect)가 고객사의 클라우드 아키텍처를 설계합니다.
🗺️PMO (프로젝트 관리 조직)PMO▼
대형 IT 프로젝트를 계획대로 완료시키는 전문 관리 조직
Project Management Office의 약자입니다. 일정, 예산, 인력, 리스크를 체계적으로 관리해 프로젝트가 목표대로 완성되도록 합니다.

🎯 비유: 건물 공사 현장 감독 — 각 팀이 제때 제대로 하는지 총괄 조율메타넷 직원의 18.3%가 PMO 역할을 수행합니다.
🔢7R 전략 (클라우드 이전 전략)7R▼
기존 시스템을 클라우드로 옮기는 7가지 방법
AWS가 정의한 마이그레이션 전략 프레임워크입니다.

Rehost: 그냥 통째로 옮김 (Lift & Shift)
Replatform: 조금만 수정해서 옮김
Repurchase: SaaS 서비스로 교체
Refactor: 클라우드 네이티브로 재설계
Retire: 안 쓰는 것 폐기
Retain: 당분간 현재 상태 유지
Relocate: 다른 클라우드로 이전

🏡 비유: 이사할 때 — 가구를 그대로 가져갈지, 새로 살지, 버릴지 결정하는 것
🤝SLA (서비스 수준 협약)SLA▼
서비스 품질 기준을 계약으로 보장하는 약속
Service Level Agreement의 약자입니다. "서버 가동률 99.9% 이상 보장", "장애 발생 시 4시간 이내 복구" 같이 서비스 품질 기준을 수치로 명확히 계약하는 것입니다.

📜 비유: 택배 "내일 배송 보장" 서비스 — 못 지키면 환불해주는 명확한 약속
🏷️TCO (총 소유 비용)TCO▼
IT 시스템을 도입·운영하는 데 드는 모든 비용의 합계
Total Cost of Ownership의 약자입니다. 장비 구매비만이 아니라 전기료, 인건비, 유지보수비, 업그레이드 비용 등을 모두 합산한 실제 비용입니다.

🚗 비유: 자동차 가격만 보면 안 되고, 기름값+보험+수리비+주차비를 합쳐야 진짜 비용클라우드로 전환하면 TCO를 30~50% 줄일 수 있다는 것이 일반적인 분석입니다.
📡Observability (관찰 가능성)OBSERVABILITY▼
시스템 내부 상태를 외부에서 얼마나 잘 파악할 수 있는가
단순 모니터링(뭔가 이상해!)을 넘어, 왜 이상한지까지 파악할 수 있는 능력입니다. 로그(기록), 메트릭(수치), 트레이스(추적)의 3가지 요소로 구성됩니다.

🩺 비유: 체온(메트릭) + 증상일지(로그) + 혈액 경로(트레이스) = 정확한 진단 가능Datadog, Dynatrace가 대표적인 Observability 플랫폼입니다.
🤖AI/ML 인프라 (GPU 서버)AI INFRA▼
AI 모델 학습과 추론을 위한 고성능 컴퓨팅 인프라
ChatGPT 같은 AI를 만들려면 엄청난 계산이 필요합니다. CPU 대신 GPU(그래픽 처리 장치)를 수백~수천 개 연결한 서버가 필요합니다.

🧮 비유: CPU = 수학 천재 1명 / GPU = 평범한 학생 수천 명이 동시에 계산 — AI에는 동시 처리가 핵심메타넷은 GPU 인프라 구축 및 AWS/Azure의 AI 클라우드 서비스를 제공합니다.
